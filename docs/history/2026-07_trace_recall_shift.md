# History: Trace Recall Shift

This project shifted from memory retrieval to trace-based recall during the AIKanojyo memory design discussions.

The key turning points were:

1. Retrieval did not feel like remembering.
2. Human memory did not feel sentence-based.
3. The circle-and-thread model emerged.
4. Weighted edges felt wrong.
5. Count-based repeated traces felt closer to experience.
6. Remembering and speaking were separated.
7. Trace Principle was defined.
8. The probe became an independent research project.
